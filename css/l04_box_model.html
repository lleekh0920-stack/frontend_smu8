블럭 요소의 박스모델

  • width*height 컨텐츠 영역(content-box)
  • padding 컨텐츠 영역의 여유 공간
  • border 컨텐츠의 테두리 (border-box)
  • margin 컨텍츠에 접근할 수 없게 하는 경계
  • **outline 문신처럼 테두리 작성(박스모델에서 제외 크기에서 제외되기 때문)
/* shortcut width style color */ border: 4px dotted blue; /* shortcut top right bottom left 시계방향*/ border-width: 2px 4px 8px;
