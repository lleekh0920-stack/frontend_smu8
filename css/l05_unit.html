단위에 대해 알아보자

  • em: 부모요소의 폰트 사이즈를 기준으로 배율지정
  • rem : 루트(html)에 지정된 폰트 사이즈를 기준으로 배율지정
  • % : 부모 요소를 기준으로 %크기 지정 주의!!( 부모 요소의 높으는 auto 로 고정되어 있지 않기 때문에 자식이 %로 높이를 지정하려면 부모의 높이가 고정이어야함 )
  • auto : 컨텐츠 크기 만큼 늘어남 (높이에 지정됨)
  • vh,vw : 화면에 보이는 브라우저의 크기 뷰포트 기준(1~100)
  • fit-content : 컨텐츠(자식)요소의 크기에 맞춤
5em*5em
5rem*5rem
20%*20%
10vw*10vh
