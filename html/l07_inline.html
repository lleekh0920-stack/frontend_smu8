인라인 요소를 알아보자!

웹 접근성(Web Accessibility)은 장애인,
고령자 등을 포함한 모든 사용자가 신체적·기술적 제한 없이 웹 사이트에서 제공하는 정보에 동등하게 접근하고 이용할 수 있도록 보장하는 것을 말합니다. 한국에서는 『디지털포용법』, 『장애인차별금지법』 등에 따라 공공기관 및 법인 웹사이트 준수가 의무화된 사항입니다. 

int sum(int a,int b){
    return a+b;
}

인라인 요소의 특징

  1. 엔터인 문자가 적용되지 않음
  2. 너비 높이는 컨텐츠 크기로 적용
  3. 이미지는 인라인블럭(box) 요소로 인라인에 포함될 수 있다.
  4. strong, em 의미론적 태그가 존재함
