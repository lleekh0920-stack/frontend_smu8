리소스를 참조하는 요소와 경로

경로 path

  • http:// , jdbc://, fs:// : 절대경로 (완전히 다른 서버로 접속시 사용)
  • / : 루트 상대경로 (같은 서버의 루트경로)
  • ./, ., "" ( img, .img, ./img) : 현재폴더를 기준으로 상대경로
  • ../ .. : 이전폴더로 이동

img

iframe : 다른 리소스를 해당 리소스(html)에 포함

a : 링크를 만드는 태그 (하이퍼텍스트)
